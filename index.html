Qu’est-ce que le Cloud Computing ?

Le Cloud Computing est la fourniture à la demande de ressources informatiques via Internet.

  • IaaS → Infrastructure (serveurs, stockage)
  • PaaS → Plateforme de développement
  • SaaS → Logiciels en ligne

Modèles IaaS / PaaS / SaaS – NIST

1

Avantages du Cloud

  • Évolutivité instantanée
  • Réduction des coûts (CapEx → OpEx)
  • Accès mondial 24/7
  • Mises à jour automatiques
  • Collaboration facilitée

Économies moyennes : 30 à 50 %

2

Inconvénients & Risques

  • Vendor lock-in
  • Factures imprévues
  • Sécurité & RGPD
  • Pannes globales
  • Souveraineté des données

Exemple : panne AWS décembre 2021

3

Conclusion

En 2025, le Cloud est devenu l’infrastructure par défaut.

Compétences à maîtriser en BTS SIO SLAM :

  • Docker & Kubernetes
  • Terraform / IaC
  • AWS ou Azure
  • Sécurité Cloud
